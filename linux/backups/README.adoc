== Liste de fichiers ou de dossiers utiles à sauvegarder

=== Cas général

Le dossier de root :

[source, bash]
----
/root/
----

Des dossiers de configuration générale :

[source, bash]
----
/etc/ssh
/etc/resolv.conf
/etc/fstab
/etc/default/grub
----

Les dossiers de configuration propre à l'utilisateur :

[source, bash]
----
/home/user/.*
----
